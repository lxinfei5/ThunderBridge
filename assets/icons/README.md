# Desktop icons (optional)

`windows\Install-DesktopIcons.ps1` will use custom icons if you drop them here:

| Shortcut | File it looks for |
|----------|-------------------|
| UltraCode (All Models) | `assets/icons/ultracode.ico` |
| Claude Code (Normal)   | `assets/icons/claude.ico` |

If a file isn't present the shortcut is still created with the default Windows
icon. Add your own `.ico` files with these names to customize.
